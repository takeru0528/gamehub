# 🈶 kanji-typing

レトロ風のUIで楽しむ、漢字タイピングゲーム！  
ユーザーが自由に漢字リストを設定できる、カスタマイズ可能な学習＆ゲーム体験。

---

## 🎮 特徴

- ⌨️ タイピングで漢字を覚える
- 🕹️ レトロスタイルのUI（ピクセルフォント、CRT風エフェクトなど）
- 📋 自分だけの漢字リストを作成可能
- 🔀 ランダム出題、Enterキー判定など快適な操作性

---
